Detailed Description

template<typename T = void>
class Mackey::Graph< T >

A directed Graph.

This is a general class that can only be used by inheriting from it. T is the type of the graph eg weightedGraph or coloredGraph. It's used for the CRTP compile-time polymorphism. By default it's void
