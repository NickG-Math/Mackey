◆ AbelianGroup() [2/2]

AbelianGroup ( const T &  a)

Given an array of numbers a[0],a[1],...,a[n] constructs the group G(a[0])+G(a[1])+...+G(a[n]) where G(1)=Z and G(n)=Z/n if n>=1 We assume that in the provided array, a[i] divides a[j] if i<=j
