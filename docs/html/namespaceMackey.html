◆ fScalar

typedef float Mackey::fScalar

The floating point type for Homology computations.

Should be float to maximize performance, or potentially double for groups of order not a prime power.

◆ pScalar

typedef short Mackey::pScalar

The integer type for the permutation matrices.

Short should be good enough for everything, char is only good for the additive structure.
