Mixes the left and right differentials to produce the differential for the box product.

The left differentials are stored in a vector L and the right differentials in a vector R. The mixing is L[0], then R[0] directly to the right, then L[1] directly below then... It's assumed that L,R have consistent dimensions for the blocks to fit coherently.
