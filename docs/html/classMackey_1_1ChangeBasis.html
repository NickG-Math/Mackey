Detailed Description

template<typename pScalar>
class Mackey::ChangeBasis< pScalar >

The change of basis permutation matrices needed for the equivariant tensor product.

The template parameter pScalar is the scalar used for the permutation vectors eg uint64_t The boolean set_conv_to_canon must be set to 1 if we want Left_to_Canon() or Right_to_Canon() The boolean set_canon_to_conv must be set to 1 if we want Canon_to_Left() or Canon_to_Right()
