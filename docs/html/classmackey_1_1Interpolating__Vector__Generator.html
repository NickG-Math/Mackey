Detailed Description

template<typename T>
class Mackey::Interpolating_Vector_Generator< T >

Generates all vectors interpolating between a given min and max vector (with the same length)

Template Parameters
TThe type of vectors eg std::vector<int>

Use with a ranged for loop: Ex for (const auto& i:v){ ... } where v is a vector_interpolate_generator object.
